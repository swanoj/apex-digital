# APEX Digital

Bespoke digital studio — Melbourne, AU.
Built with Next.js 15 + TypeScript + Tailwind CSS.
Hosted on Vercel. 

## Stack
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS + CSS Variables
- **Content**: Markdown files (no database)
- **Email**: Resend
- **Hosting**: Vercel

## Getting started
```bash
npm install
npm run dev
```

## Adding a project
Create a new `.md` file in `content/projects/` — copy an existing one as a template.

## Environment variables
Copy `.env.example` to `.env.local` and fill in your values.

## Deploying
Push to `main` — Vercel auto-deploys in ~30 seconds.
